# HANAPrometheusGrafana
Prometheus metrics and Grafana samples dashboards for SAP HANA

## Prerequisites

[SUSE HANA db exporter](https://github.com/SUSE/hanadb_exporter)

[SUSE Host Agent Tool](https://github.com/SUSE/shaptools)

[Prometheus Python client](https://github.com/prometheus/client_python)

[HANA Python client](https://help.sap.com/docs/SAP_HANA_PLATFORM/0eec0d68141541d1b07893a39944924e/39eca89d94ca464ca52385ad50fc7dea.html)

[Prometheus](https://prometheus.io/download/)

[Grafana](https://grafana.com/)

[Intel Process Counter Monitor PCM](https://github.com/opcm/pcm)
